# magic-cube-7x7x7
Основные характеристики: 
1. Порядок куба: 7×7×7
2. Числа: от 1 до 343 (7³)
3. Магическая константа: M = 1204

## 🔑 Ключевые детали:
Аутентичные данные: Числа взяты из таблиц 1–4 оригинальной [статьи](https://www.trump.de/magic-squares/magic-cubes/New-Discoveries-in-the-History-of-Magic-Cubes.pdf), а не сгенерированы «на глаз».

Корректная индексация: Используется формат `cube[z][y][x]`, где:

 - z — номер слоя (0–6),
 - y — строка (сверху вниз),
 - x — колонка (слева направо).

Формула де Фибра: Добавлена функция `de_fibre_formula_7x7x7()`, реализующая математическую конструкцию из [статьи](https://www.trump.de):

`w = 49·c + 7·b + a + 1, где (a,b,c)ᵀ = M·(x,y,z)ᵀ + t mod 7`

### Полная верификация: Проверяются не только базовые суммы, но и все 42 диагонали сечений + 4 пространственные диагонали.
