# magic-cube-7x7x7
Основные характеристики: 
1. Порядок куба: 7×7×7
2. Числа: от 1 до 343 (7³)
3. Магическая константа: M = 1204

## 🔑 Ключевые исправления:
Аутентичные данные: Числа взяты из таблиц 1–4 оригинальной [статьи](www.trump.de), а не сгенерированы «на глаз».

Корректная индексация: Используется формат cube[z][y][x], где:

  z — номер слоя (0–6),
  y — строка (сверху вниз),
  x — колонка (слева направо).

Формула де Фибра: Добавлена функция de_fibre_formula_7x7x7(), реализующая математическую конструкцию из [статьи](www.trump.de):
`w = 49·c + 7·b + a + 1, где (a,b,c)ᵀ = M·(x,y,z)ᵀ + t mod 7`

### Полная верификация: Проверяются не только базовые суммы, но и все 42 диагонали сечений + 4 пространственные диагонали.

▶️ В результате запуска вам будет показан гарантированно валидный совершенный магический куб 7×7×7, готовый для визуализации и поиска геометрических паттернов.
